Propstack-CRM-Integration für RESA — synchronisiert Leads automatisch.

== Description ==

RESA für Propstack ist ein Premium-Add-on für das RESA-Plugin, das Lead-Daten automatisch mit Propstack CRM synchronisiert.

**Features:**

* Automatische Lead-Synchronisation zu Propstack CRM
* Makler-Zuweisung pro Standort
* Aktivitäten und Aufgaben erstellen
* Newsletter-DOI-Integration
* Kontaktquellen-Zuordnung
* Detailliertes Sync-Status-Tracking

**Voraussetzungen:**

* RESA Plugin (Premium-Lizenz erforderlich)
* Propstack CRM Account mit API-Zugang

== Installation ==

1. RESA Premium muss installiert und aktiviert sein
2. ZIP-Datei hochladen und aktivieren
3. Unter RESA → Integrationen → Propstack konfigurieren
4. API-Key eingeben und Verbindung testen
