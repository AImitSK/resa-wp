Interaktive, leadgenerierende Rechner und Tools für Immobilienmakler im DACH-Raum.

== Description ==

RESA bietet interaktive Rechner und Tools, die Immobilienmakler auf ihrer Website einbetten können, um qualifizierte Leads zu generieren.

**Enthaltene Tools (Free):**

* Mietpreis-Kalkulator
* Immobilienwert-Kalkulator

**Pro-Tools:**

* Kaufnebenkosten-Rechner
* Budgetrechner
* Renditerechner
* Energieeffizienz-Check
* Verkäufer-Checkliste
* Käufer-Checkliste

== Installation ==

1. Plugin hochladen und aktivieren
2. Unter RESA im Admin-Menü die gewünschten Tools konfigurieren
3. Shortcode `[resa]` auf der gewünschten Seite einbinden
